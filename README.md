Caso de Estudio para el Portafolio
------------------------------------

Nombre de la Sección: Deep Dive: Ingeniería Frontend

  Proyecto. 
      "Vue Product Showcase con Gestión de Estado"
      Descripción: Una Single Page Application (SPA) para un catálogo de productos que consume una API externa y permite gestionar un carrito de compras.

  Desafío Principal.
      Centralizar el flujo de datos para evitar el "prop drilling" y asegurar que la interfaz reaccionara en tiempo real a los cambios del usuario.

  Solución.
      Implementación de Vuex para el manejo de estado global y Axios para la comunicación asíncrona con el backend.

  Herramientas.
      Vue.js, Vuex, Vue Router, Vitest (para pruebas unitarias) y SASS.

  Aprendizajes.
      Control de ciclos de vida en componentes de Vue y la importancia de las pruebas automatizadas para prevenir errores en producción.

  Métricas de Impacto.
      Reducción del 40% en líneas de código redundantes gracias a la centralización del estado y cobertura de tests del 80% en funciones críticas.

  Habilidades Aplicadas.
      Lógica de programación en JS, arquitectura de componentes, y diseño responsive con Flexbox/Grid.

  Justificación de elección.
      Elegí este proyecto porque demuestra mi capacidad para manejar flujos de datos complejos y mi compromiso con el código limpio y testeado, superando el nivel básico de HTML/CSS.
